# 📝 To-Do App

A simple and interactive **To-Do App** built using **Vanilla JavaScript**.
This project helps users manage daily tasks efficiently.

---

## 🚀 Features

* ➕ Add new tasks
* ✅ Mark tasks as completed
* ❌ Delete tasks
* 📌 Simple and clean UI
* ⚡ Fast and lightweight (no frameworks)

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript (ES6)

---

## 📂 Project Structure

```
todo-app/
│── index.html
│── style.css
│── script.js
│── README.md
```

---

## ▶️ How to Run

1. Download or clone this repository
2. Open the project folder
3. Double-click on `index.html`

👉 The app will open in your browser
